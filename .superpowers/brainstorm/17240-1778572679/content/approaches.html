Injecting the "Impeccable" Guidelines into the Swarm

How should the bees apply their newfound design skills?

A

The Prompt Injection (Recommended)

We inject the strict "Impeccable" design laws directly into the Gemma 4 system prompt in the API route. The LLM acts as the design reviewer, ensuring it only outputs CSS hacks that follow brutalist OKLCH coloring, high-contrast typography ratios, and no cheap gradients.

B

The Filter / Validator

The AI proposes whatever chaos it wants, but before injecting it into the DOM, we run it through an "Impeccable Filter" in TypeScript that automatically transforms bad colors to OKLCH and removes ugly fonts, acting like a strict art director.

C

The Pre-Curated Design System

The AI doesn't generate raw CSS variables anymore. Instead, it just selects from a list of 5-10 pre-curated "Impeccable" design tokens (e.g. `theme_hack: 'brutalist-monochrome'`), and the local React app applies the corresponding pre-written CSS.
