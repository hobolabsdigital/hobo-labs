Design: Impeccable Bees via Prompt Injection

Here's how we'll upgrade the swarm's brain.

1. The New AI Persona (api/bee/decide/route.ts)

We will rewrite the Gemma 4 system prompt to include core Impeccable Design Laws:

  • Color: Use OKLCH. Never use pure #000 or #fff. Tint every neutral toward the brand hue.
  • Typography: Ensure strong hierarchy through scale and weight contrast (≥1.25 ratio). Cap line lengths to 65-75ch.
  • Aesthetic: Strict bans on side-stripe borders, gradient text, and default glassmorphism. Push for bold, editorial, magazine-like layouts.
2. The Target Mechanics (Swarm.tsx & useBeeStore.ts)

We will explicitly instruct the AI to target specific nodes. The AI will output a JSON payload like this:

{
  "mischief": "theme_hack",
  "targetNodeId": "hero-1",
  "cssVars": "--bg: oklch(25% 0.05 250); --fg: oklch(95% 0.01 250);"
}

The Swarm will read this, fly to hero-1, and inject the variables scoped only to that node.

3. Node Implementation

We will ensure that the React Flow node components (like HeroNode and TextNode) actually use the injected CSS variables for their background colors, text colors, and fonts.
