## The Problem
Passive digital consumption is directly causing developmental delays and reducing expressive language skills in young children.

## The Solution
An immersive 3D open-world educational game featuring a real-time Speech-to-Intent pipeline. Children solve quests and direct 'Monster Movies' using their own voice, supported by local on-device transcription models (Qwen).

## Overview & Impact
MonstoryX turns speech into interactive stories, promoting literacy, empathy, and computational thinking. As Co-Founder, I engineered the Voice-To-Intent pipeline and the proprietary `MonstoryReplaySystem` for the Movie Maker mode using custom C++ plugins, with backend multiplayer logic handled via Nakama. We also architected an innovative physical-to-digital asset pipeline: characters were hand-sculpted in Plasticine, processed through Hunyuan 3D AI, and rigged in UE5. This project proves that high-fidelity gaming technology can be leveraged to forge deep, educational human connections.
