# RISE ePA+ (Electronic Patient Record)

## Overview

RISE ePA+ is an advanced digital health application that extends the standard German Electronic Patient Record (ePA) with powerful user-centric features. It modernizes the healthcare system by safely connecting patients with doctors, pharmacies, and insurance providers while offering intelligent health management tools directly on their smartphones.

## Tech Stack

- iOS Native
- Swift & SwiftUI
- gematik Security Standards Integrations

## My Role & Contributions

- **iOS Developer:** Core contributor to the native iOS application development.
- **Feature Engineering (SwiftUI):** Designed, developed, and maintained complex UI architectures and workflows utilizing modern SwiftUI, ensuring an accessible, highly responsive, and strictly secure patient user experience.

## Interesting Details

- **Intelligent Health Features:** The ePA+ ecosystem goes far beyond basic document storage. It includes interactive, natively built sub-modules like a comprehensive Health Diary (e.g., tracking migraines), a Preventive Care Manager, and dynamic Vaccination Recommendations.
- **Patient Empowerment:** Designed to encourage active patient participation and adherence (including rehab tracking) by bringing complex medical data into a fluid, easy-to-understand, mobile-first interface.
- **Top-Tier Security:** Because it handles highly sensitive medical data, all features were rigorously engineered to meet the strict regulatory and security specifications mandated by 'gematik'.
