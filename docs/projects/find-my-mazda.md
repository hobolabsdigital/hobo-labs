# Find My Mazda (Mazda Motors Germany)

## Overview

An entertaining, personalized sales platform created for Mazda Motors Germany to help customers find their ideal vehicle. "Find My Mazda" is an intuitive online hub that connects inspiration and purchase, combining real-time availability checks, location-based searching, financing configuration, and direct test-drive bookings into a seamless experience.

## Tech Stack

- Vue.js (frontend bundled for native Mazda website embedding)
- Elastic Search Backend
- Node.js

## My Role & Contributions

- **Lead Developer:** Architected and developed the platform in collaboration with Demodern.de for Mazda.
- **Micro-Frontend Architecture:** Engineered the Vue application so it could be strictly bundled and seamlessly embedded directly into the existing Mazda.de infrastructure.
- **Search System:** Integrated a lightning-fast Elastic Search backend to handle real-time inventory filtering and location-based matching.

## Interesting Details

- **Conversational UX:** Replaced traditional sterile search filters with a playful dialogue. Six short questions gracefully guide users like a conversation to find the perfect model.
- **Real-Time Dynamic Results:** Search results update instantaneously with every input, showcasing accessory customizations and exactly which vehicles are locally available with precise pricing and offers.
- **Massive KPI Impact:** Following the launch, the platform achieved an 18% increase in overall conversion rate and a 30% rise in highly-qualified dealer inquiries. Dealers specifically praised the high relevance of leads, establishing the tool as a benchmark for customer-oriented online automotive selling.
