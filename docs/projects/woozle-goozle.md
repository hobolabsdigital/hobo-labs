## The Problem
Transforming a beloved physical TV hand puppet into a fully interactive digital companion required bypassing traditional static UI to create an authentic conversational bond with children.

## The Solution
An interactive React Native application featuring autonomous voice control via TensorFlow and Dialogflow, seamlessly layered over a Unity-driven virtual 3D environment.

## Overview & Impact
Responsible for the React Native UI components, video logic, and overall application flow control while providing Unity support to integrate the virtual environment. Transitioned the physical hand puppet into a digital 3D character using custom furless shaders, painstakingly imitating original movements. Created a custom dialogue database matching ~1250 unique keywords to over 650 specifically animated sequences, allowing for an ever-expanding conversational experience with seamless lip syncing.
