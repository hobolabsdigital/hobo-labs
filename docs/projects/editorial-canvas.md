## The Problem
Traditional portfolios are static, rigid, and ultimately fail to demonstrate true AI-native capabilities or agentic coding paradigms in real-time.

## The Solution
Built an interactive, brutalist canvas using React Flow, where a local Ollama model orchestrates autonomous node spawning, layout arrangement, and physics based entirely on fluid user conversation.

## Overview & Impact
This website serves as a live, interactive demonstration of 'Agentic Coding.' By deeply integrating the Vercel AI SDK with custom semantic RAG retrieval, the portfolio acts as a fully-featured digital twin. It is capable of autonomously rendering rich project nodes, hero text, and complex spatial layouts—all wrapped in a highly-polished, brutalist design aesthetic that bridges human interaction and machine orchestration.
