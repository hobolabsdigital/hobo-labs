# StopOverfishing (Oceana)

## Overview

An interactive web experience created for the environmental organization Oceana to make the issue of overfishing visible. Users design their own low-poly fish with a personal message, and every shared social media post adds a unique fish to a real-time, lively virtual ocean, turning individual engagement into a massive visible swarm.

## Tech Stack

- React & Three.js (WebGL)
- Node.js & PostgreSQL
- Python (Twitter Scraping Service)
- AWS

## My Role & Contributions

- **Web Developer & Creative Technologist:** Architected and developed the interactive web experience from research and prototyping to launch.
- **Real-Time Swarm Rendering:** Engineered the performant Three.js underwater world, rendering massive swarms of low-poly fish with vibrant colors and natural movements across various devices.
- **Social Media Synchronization:** Built a robust Python and Node.js backend pipeline that continuously scraped Oceana hashtags in real-time and synchronized new entries via AWS cloud solutions to update the client-side swarm instantly.

## Interesting Details

- **The Fish Creator:** Playfully designed tool allowing users to customize colorful, low-poly fish that carry their personal conservation message.
- **Real-Time Synchronized Ocean:** A cloud service monitors hashtags and updates all active visitors simultaneously, letting everyone watch the virtual ocean grow in real-time as the campaign spreads.
- **Accessible Low-Poly Aesthetics:** Natural swarm movements and vibrant aesthetics created an emotional connection, making the serious topic of overfishing tangible and inviting without feeling explicitly technical.
